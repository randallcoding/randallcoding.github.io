Hey!

My name is Alex Randall. I am currently a student at iO Academy's January 2023 cohort. I've always been interested in learning programming, so I decided to take a dive head first into the bootcamp. If there's one major thing I've learned, it's that I love the logical puzzles coding brings. Below, you'll find the various projects I and my team have been working on...

PHP Collector App

In this project, we were tasked with making an SQL database containing items that we like. We then wrote a collection app in PHP to fetch the database and display the items on a website.

Javascript Game

In this project we were tasked with making a JavaScript game that had you guess UK counties within a time limit. We learned how to effectively use Git as a team and how to utilise CSS to make the front end more accessible to the player.

React App

A restaurant app built in React consuming a pre-built API to fit a supplied design. It was styled using Sass and Bootstrap, and utilises modern JavaScript practices such as fetch to interact with the back-end API and hooks to handle the rendering of the data.

RESTful API

In this project, we were tasked with building an API conforming to the RESTful architectural style and SOLID practises.

Student Management App

With this project, it was a chance to further apply our PHP skills as we were tasked with working on an existing legacy code base - iO Academy's AcademyPortal. For this project we were required to add extra functionality and fix some bugs. We used PHP, MVC, Slim and MySQL for the back-end and data was displayed to the front-end using JavaScript and HTML. We added a feature where it would count the number of places left on a course, and did bug fixes on the applicant CSV uploader.
